# 📚 Book Review App

A full-stack Book Review web application built using the **MERN stack** – MongoDB, Express.js, React, and Node.js. This app enables users to explore books, filter them, view detailed information, and submit reviews.

## 🚀 Features

### 1. 📖 Book List Page

- **Navbar**: Includes search bar and profile icon
- **Sidebar**: Filter books by genre or category
- **Product List**: Paginated display of books

![Book List Page](./assets/book-list.png)

---

### 2. 🧑‍💼 Dashboard

- **User Profile**: View profile information
- **Add New Book**: Button to create and submit a new book entry

![Dashboard](./assets/dashboard.png)

---

### 3. 📘 Book Details Page

- **Book Info**: Title, author, description, etc.
- **Reviews**: List of reviews with reviewer names and comments
- **Add Review**: Users can submit a new review

![Book Details Page](./assets/book-details.png)

---

## 🛠️ Tech Stack

- **Frontend**: React, React Router, CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Other Tools**: Axios, dotenv, nodemon
